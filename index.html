SolidWorksKinematicsKutzbach Criterion Vector Loop AnalysisMATLABGear Design3D Printing
Mechanism Analysis
Verified 1-DOF mobility using the Kutzbach criterion: m = 3(n−1) − 2j₁ − j₂. With n=5 links and j₁=6 full joints, the raw result is m=0 — but meshing gears are an exception that subtract one additional DOF, giving actual mobility m=1. Position and velocity analysis used MATLAB vector loop equations solved via fsolve; acceleration analysis derived kinematic coefficients from the Jacobian.
